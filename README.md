# ESP32-
本專題開發一套基於 ESP32 的「自定義智慧氣象數據排版系統」。系統核心在於解決傳統顯示設備介面固定的問題，利用 ESP32 建立 區域網路 Web Server，讓使用者無需重新燒錄程式，即可透過手機瀏覽器即時調配 OLED 螢幕上的數據佈局。  技術層面透過 HTTPClient 串接中央氣象署（CWA）開放資料 API，獲取精確的在地化溫度、濕度與風速，並結合 NTP 伺服器 進行高精度時間同步。系統採用物件化數據映射邏輯，將純數據與顯示標籤分離，實現「極簡化」的視覺風格。本應用不僅展示了物聯網（IoT）數據處理與動態排版之整合，更提供了一個低成本且具備高度客製化能力的智慧家居解決方案。
